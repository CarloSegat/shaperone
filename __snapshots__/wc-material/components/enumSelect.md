# `wc-material/components/enumSelect`

#### `renders an mwc-select`

```html
<mwc-select>
  <mwc-list-item
    aria-disabled="false"
    mwc-list-item=""
    role="option"
    tabindex="0"
    value="foo"
  >
    foo
  </mwc-list-item>
  <mwc-list-item
    aria-disabled="false"
    mwc-list-item=""
    role="option"
    tabindex="-1"
    value="bar"
  >
    bar
  </mwc-list-item>
</mwc-select>

```

#### `sets selection to current object`

```html
<mwc-select>
  <mwc-list-item
    aria-disabled="false"
    mwc-list-item=""
    role="option"
    tabindex="0"
    value="foo"
  >
    foo
  </mwc-list-item>
  <mwc-list-item
    aria-disabled="false"
    aria-selected="true"
    mwc-list-item=""
    role="option"
    selected=""
    tabindex="-1"
    value="bar"
  >
    bar
  </mwc-list-item>
</mwc-select>

```
